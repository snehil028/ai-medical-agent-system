## Overview
This project implements a multi-agent AI system for analyzing medical reports. Multiple specialized agents process the same input in parallel and generate structured insights, which are then aggregated into a final explainable output.

## Key Features
- Multi-agent architecture simulating a multidisciplinary medical board (cardiology, psychology, pulmonology)  
- Parallel execution using Python threading for faster inference  
- Aggregation layer to combine outputs into structured diagnostic insights  
- Modular design enabling easy addition of new agents
